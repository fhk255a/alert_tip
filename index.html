Alert 插件演示

当前使用的是修改版
初试版 AlertCss.min下载 | AlertJS.min下载
修改版 AlertCss下载 | AlertJS下载

用法:

1. 引入配置的css

< script src= " js/jquery.min.js " > < /script >

2. 引入Jquery

< link rel= " stylesheet " href=" AlertCss.min.css " >

3. 引入该JS

< script src= " AlertJS.min.js "> < /script >

HTML

< button class="btn" > Click Me < /button >

JavaScript

例如:

$('btn').click(function () {

new Alert( {

type:success

text:'这是一段小句子'

button:'点赞'

title:'精选句子'

move:0

} );

});

使用说明

参数 类型 值 说明
type string success | error | warn 提示的效果
title string '标题' 标题
text string '内容' 内容
button string '确定' 按钮的值,只能一个的时候使用
background string 'rgba()' | '#ffa' | 'red' 弹框背景颜色
buttons arr ['点赞','踩','转发'] 按钮组< 数组有多少个 按钮就对应多少个 最多四个 >
input object {type:'text',value:'搜索','background':'red'} input对象组
move number 0 | 1 | 2 | 3 运动效果,只有4种

注意事项

该插件为初试版本,可能存在兼容性问题,与bootstrap可能存在一些小bug.

如果跟Bootstrap一起使用,请使用修改版

否则用初始版本
